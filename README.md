# KimiBoard
KimiBoardは「置き型」「キーボード埋め込み型」の2タイプの使い方が出来るトラックボールマウスです。）

| 名前 | 仕様 |
|:-|:-|
| KimiBoard      | 3キー + トラックボール|
| KimiBoard Plus      | 6キー + トラックボール|
| KimiBoard Long| 6キー + トラックボール + ロータリーエンコーダー x 3|


【特徴】
- 有線・無線両対応
- 専用のアタッチメントを使うことによりUSキーボードのスペースキー部分に埋め込むことが可能
- 各キーはZMK Firmwareにてカスタマイズ可能
- 20mmトラックボール採用
